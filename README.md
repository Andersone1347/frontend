## Урок 3.
Создание новых  папок **css** и **img**



Использование переменных в css

Чтобы вызывать переменные, нужно выполнить  след. шаги :
- объявить переменную 
```
--gutter: 'значение';
```
- поместить  переменные в файлс css в root 
```
:root {
 --gutter: 20px;
 --bg: tomato;
}
```
